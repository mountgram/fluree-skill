# fluree-skill

Agent skill for Fluree DB v4 CLI operations, RDF/Turtle and JSON-LD data modeling, SPARQL and JSON-LD queries, transactions, branching, time travel, full-text search, and graph schema design.

## Install

```bash
npx skills install https://github.com/mountgram/fluree-skill/tree/main/skills/fluree
```

## Fluree Docs

Fluree DB v4 changes quickly, so the skill points agents back to the live docs for current command syntax and behavior:

- Fluree DB docs: https://labs.flur.ee/docs/db
- CLI reference: https://labs.flur.ee/docs/db/cli
- `fluree init`: https://labs.flur.ee/docs/db/cli/init

## Layout

The skill lives at `skills/fluree/SKILL.md` using the Agent Skills specification layout.
