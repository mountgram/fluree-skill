# fluree-skill

Agent skill for Fluree DB v4 CLI operations, RDF/Turtle and JSON-LD data modeling, SPARQL and JSON-LD queries, transactions, branching, time travel, full-text search, and graph schema design.

## Install

```bash
gh skill install mountgram/fluree-skill fluree --agent opencode --scope project
```

## Layout

The skill lives at `skills/fluree/SKILL.md` using the Agent Skills specification layout.
